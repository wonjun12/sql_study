화면 기획서

- foreign 키를 지정할때 다른 테이블들의 다수를 지정할때
  테이블 이름_지정할 id 형식으로 이름을 정한다.
   - 다른 테이블에서 외래키로 같은 이름을 사용하는 경우가 많기 때문에 테이블 이름을 넣어준다.
    예시) comment_photo_id
            comment_user_id

 - 기본키를 2개이상 해야하는 경우가 있다.
  - 좋아요 같은 시스템인데, 기본키를 2개이상 해야할때는 id 컬럼이 필요없다.

 - workbench에서 프로그램으로 만든 테이블에 이나 데이터베이스에 대해 코드를 복사 할 수있다.
   -> Copy to Clipboard -> Create ... 등 누르면 코드가 복사됨
